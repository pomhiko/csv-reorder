# CSV列整理くん

## 概要

CSV列整理くんは、CSVの列名変更・列並び替え・列追加・列削除をブラウザ上で行えるシンプルなCSV整形ツールです。

EC運営、在庫管理、倉庫提出用CSV、バックオフィス業務などで発生しがちな「提出先ごとにCSVの列順やヘッダー名が違う」問題を、1画面で素早く解消することを目的にしています。

## 特徴

- CSVファイルをブラウザ上で読み込み
- 1行目をヘッダー行として認識
- ヘッダー名をinputで編集
- ドラッグ&ドロップで列を並び替え
- 不要な列を削除
- 削除済み列を一覧から復元
- 空の列を末尾に追加
- 先頭5行のプレビュー表示
- 変換後CSVをダウンロード
- サンプルCSVをダウンロード
- CSV以外、空CSV、ヘッダーなしCSVのエラー表示
- 重複ヘッダーの警告と読み込み選択

## 使い方

1. CSVファイルをアップロードします。
2. 必要に応じてヘッダー名を編集します。
3. 列カード左側のハンドルをドラッグして列順を変更します。
4. 必要に応じて列を追加・削除・復元します。
5. プレビューで内容を確認します。
6. 「変換後CSVをダウンロード」からCSVを書き出します。

## 対応していること

- CSVファイルのアップロード
- CSVの1行目をヘッダーとして読み取り
- ヘッダー名変更
- 列順変更
- 列追加
- 列削除
- 削除済み列の復元
- 並び替え・削除・追加を反映したCSV出力
- UTF-8 CSVのダウンロード
- ブラウザ内でのファイル処理

現在はCSVのみ対応しています。Excelファイルの読み込み・編集には対応していません。

## 安全性

ファイルはブラウザ内で処理され、サーバーには保存されません。

このMVPではログイン、データベース、クラウド保存、課金機能は実装していません。アップロードしたCSVの内容は、変換処理のためにブラウザ上でのみ扱われます。

## ローカル起動方法

```bash
npm install
npm run dev
```

起動後、ブラウザで以下を開きます。

```text
http://localhost:3000
```

本番ビルドを確認する場合は以下を実行します。

```bash
npm run build
```
